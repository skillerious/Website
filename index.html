Markiva v3.1 Transformation

Formerly built in Python, Markiva now leverages Electron for a truly cross‑platform markdown editing experience. This transformation supercharges performance, stability, and the UI’s polish.

Directly from the README:

  • 🚀 Major transformation from Python to Electron
  • 🚀 Cross‑platform – Windows, macOS, and Linux
  • 🚀 Polished Markdown Editing – modern interface and advanced features

Markiva’s new Electron foundation makes it easier to expand features, integrate with other services, and deliver a consistent look and feel across all desktop environments.
