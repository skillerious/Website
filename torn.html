Install Tampermonkey & Scripts for Torn

Super‑quick Chrome walk‑through for the faction crew
(Edge, Brave & Firefox work the same way)

Step 1 — Add Tampermonkey

 Get the extension

Choose your browser below, then:

  • Click Add extension (Chrome/Edge) or Add to Firefox.
  • Wait for the “Tampermonkey has been added” confirmation.
  • (Optional) Pin the Tampermonkey icon via the puzzle‑piece menu so it stays visible.
 Chrome  Edge  Firefox
https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo

Step 2 — Browse Script Gallery

 Find your tools

Head to Greasy Fork’s Torn section and:

  • Use the search bar for keywords like “revive”, “egg”, “stats”.
  • Click Sort › Last updated to see actively maintained scripts first.
  • Click a script title to open its details page.
 Open GreasyFork
https://greasyfork.org/en/scripts/by-site/torn.com

Step 3 — One‑Click Install

 Install a script

On any script page:

  • Press the big green Install this script.
  • Tampermonkey opens a preview — glance at the code if you wish.
  • Click Install (top‑right) → tab closes.

Step 4 — Enable & Verify

 Dashboard check

  • Click the Tampermonkey icon → Dashboard.
  • Ensure your new script shows a green toggle.
  • Reload Torn — most scripts add buttons/banners immediately.

Step 5 — Manage Like a Pro

FAQ / Troubleshooting

Script won’t install

Confirm the Tampermonkey extension is enabled and its icon is pinned & visible.

⚠ Update failed

GreasyFork may be down — wait a bit then try again.

No effect on Torn

Make sure you’re on https://www.torn.com/ (not the app) and the script is enabled.

Chrome blocks installation

Check extension policies in chrome://extensions — some workplaces lock installs.
